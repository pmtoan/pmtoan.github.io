Install and configure Redmine on Ubuntu 18.04
1. Install Ruby

$ sudo apt-get install curl mysql-server libmysqlclient-dev git subversion imagemagick libmagickwand-dev libcurl4-openssl-dev
$ sudo apt-get update
$ command curl -sSL https://rvm.io/mpapis.asc | gpg --import -
$ curl -L https://get.rvm.io | sudo bash -s stable
$ sudo su
# source /usr/local/rvm/scripts/rvm
# echo '[[ -s "/usr/local/rvm/scripts/rvm" ]] && source "/usr/local/rvm/scripts/rvm"' >> ~/.bashrc
# command curl -sSL https://rvm.io/mpapis.asc | gpg --import -
# rvm get head
# rvm install ruby-2.3

2. Install Passenger and Nginx

$ sudo apt-get install -y dirmngr gnupg
$ sudo apt-key adv --keyserver hkp://keyserver.ubuntu.com:80 --recv-keys 561F9B9CAC40B2F7
$ sudo apt-get install -y apt-transport-https ca-certificates
$ sudo sh -c 'echo deb https://oss-binaries.phusionpassenger.com/apt/passenger bionic main > /etc/apt/sources.list.d/passenger.list'
$ sudo apt-get update
$ sudo apt-get install -y libnginx-mod-http-passenger nginx-extras

3. Enable Passenger Nginx module

$ if [ ! -f /etc/nginx/modules-enabled/50-mod-http-passenger.conf ]; then sudo ln -s /usr/share/nginx/modules-available/mod-http-passenger.load /etc/nginx/modules-enabled/50-mod-http-passenger.conf; fi
$ sudo ls /etc/nginx/conf.d/mod-http-passenger.conf

If you don’t see a file at /etc/nginx/conf.d/mod-http-passenger.conf; then you need to create it yourself and set the passenger_ruby and passenger_root config options. For example:

passenger_root /usr/lib/ruby/vendor_ruby/phusion_passenger/locations.ini;
passenger_ruby /usr/bin/passenger_free_ruby;

4. Install and configure MySQL

CREATE DATABASE redmine CHARACTER SET utf8mb4;
GRANT ALL ON redmine.* TO 'redmine'@'localhost' IDENTIFIED BY 'change-with-strong-password';
FLUSH PRIVILEGES;
EXIT;

5. Install Redmine

$ sudo apt install build-essential libmysqlclient-dev imagemagick libmagickwand-dev ruby2.5-dev
$ cd /home/ubuntu
$ wget https://www.redmine.org/releases/redmine-4.0.3.tar.gz
$ tar -zxf redmine-4.0.3.tar.gz
$ mv redmine-4.0.3 redmine

6. Configure Redmine production database

$ cd /home/ubuntu/redmine
$ cp config/database.yml.example config/database.yml

Configure file /home/ubuntu/redmine/config/database.yml

production:
    adapter: mysql2
    database: redmine
    host: localhost
    username: redmine
    password: "change-with-strong-password"
    encoding: utf8

7. Install Ruby dependencies

$ cd /home/ubuntu/redmine
$ sudo gem install bundler --no-rdoc --no-ri
$ sudo bundle install --without development test postgresql sqlite

8. Generate key and migrate database

$ cd /home/ubuntu
$ sudo bundle exec rake generate_secret_token
$ sudo RAILS_ENV=production bundle exec rake db:migrate

9. Configure Nginx

Edit file /etc/nginx/sites-available/default

server {
    listen 80;
    server_name _;
    root /home/ubuntu/redmine/public;
    passenger_enabled on;
    passenger_min_instances 1;
    client_max_body_size 10m;
}

$ sudo ln -s /etc/nginx/sites-available/default /etc/nginx/sites-enabled/
$ sudo systemctl restart nginx

10. Go to web interface

Login into Redmine (http://localhost) with credentials: admin/admin

11. Install plugins

Go to redmine root folder
Extract plugin into plugins folder

Install dependencies

$ cd /home/ubuntu/redmine
$ sudo bundle install --without development test --no-deployment

Migrate database

$ sudo bundle exec rake redmine:plugins NAME=redmine_contacts RAILS_ENV=production

Restart redmine

$ sudo touch tmp/restart.txt
